Run manu2print on your machine - do each step in order.

Step 1: Open PowerShell (Windows key, type powershell, press Enter).

Step 2: Go to the project folder. Type: cd Desktop\layoutforge
Then press Enter. (If the project is elsewhere, use that path.)

Step 3: Create .env.local for your API key. Type: notepad .env.local
Press Enter. If Notepad asks to create a new file, click Yes.
Add one line: ANTHROPIC_API_KEY=your_key_here
Save (Ctrl+S) and close Notepad.

Step 4: Install dependencies (first time only). Type: npm install
Press Enter. Wait until it finishes.

Step 5: Start the app. Type: npm run dev
Press Enter. Wait until you see "Ready on http://localhost:3000".

Step 6: Open your browser and go to: http://localhost:3000

To stop the app: In the PowerShell window press Ctrl+C.

If "npm" is not recognized, install Node.js from nodejs.org and try again.
