# Current Limitations

- manu2print currently supports print-ready PDF files only.
- EPUB export is not supported.
- Direct publishing to KDP or other platforms is not available.
- File uploads must be under the platform size limit.
- Cover generation is manual; only calculators exist for cover/spine dimensions.
- No unified metadata manager exists for title, subtitle, or author information.
