# Troubleshooting

## PDF Upload Issues

**Possible causes:**
- File too large
- Incorrect PDF version
- Embedded fonts missing

**Solutions:**
1. Export the file again as a standard PDF
2. Ensure images are around 300 DPI
3. Upload the new file
