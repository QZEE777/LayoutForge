# Journals, coloring books, puzzle books — tool coverage

Short summary of what we offer for **kids books**, **journals/workbooks**, **coloring books**, and **puzzle/activity books**.

---

## Coverage

| Segment | Tool | What it does |
|--------|------|----------------|
| **Kids books** | [Kids book trim guide](/kids-trim-guide) | Trim sizes and page counts for picture books and children’s titles; bleed & safe zone; links to cover/spine calculators. |
| **Journals & workbooks** | [Journals, coloring & puzzle guide](/journals-coloring-puzzle-guide) | Trim sizes (8.5×11, 7×10, 6×9), page count notes, link to interior template and cover calculator. |
| **Coloring books** | Same guide | Trim sizes (8.5×8.5, 8.5×11, 8×10), bleed, 300 DPI, color interior note. |
| **Puzzle / activity books** | Same guide | Trim sizes, page counts, when to use bleed. |

All of the above are **free** reference tools. The **KDP Formatter** (DOCX/PDF) and **KDP PDF Checker** support these book types via `kdpConfig` (e.g. book type: Workbook/Journal, Coloring Book, Puzzle Book/Activity Book); the guides give format-specific tips.

---

## Possible future additions

- **Children’s / color PDF checker variant** — Same as current checker plus image DPI, bleed, and safe-zone hints (see `docs/COLOR-AND-CHILDRENS-BOOKS.md`).
- **Format Review “children’s” or “coloring” mode** — AI checks for image-heavy layout, 300 DPI, bleed, safe margins.

For now, the two guides (kids + journals/coloring/puzzle) cover the main “how do I size and format this?” questions for these segments.
